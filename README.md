## Silver-waffle

Tool which allows admin to manage images with tooltips. Admin is able to see list of images, add image, edit image, delete image. Also available editing tooltips for images.

## Tools

- React,
- react-router
- Redux
- redux-promise-middleware
- ESLint (airbnb code style)
